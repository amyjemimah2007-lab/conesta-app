# 🤖 AI Habit Coach

## 📌 Project Description

AI Habit Coach is a web application that helps users build better habits using OpenAI's language model. Users enter a habit they want to develop, and the application generates personalized motivation, practical advice, and a daily challenge.

---

## 🚀 Features

* AI-powered habit coaching
* Personalized motivational responses
* Practical daily tips
* Daily habit challenge
* Responsive and user-friendly interface
* Secure API key management using environment variables

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* Vercel Serverless Functions
* OpenAI Node.js SDK

## ⚙️ Installation

1. Clone the repository.

2. Install dependencies:

```bash
npm install
```

3. Create a `.env` file in the project root.

```
OPENAI_API_KEY=your_openai_api_key
```

4. Run or deploy the project on Vercel.

---

## ▶️ Usage

1. Open the application.
2. Enter a habit you want to build.
3. Click **Get AI Advice**.
4. Receive personalized motivation, a practical tip, and a daily challenge from the AI.

---

## 🔒 Security

The OpenAI API key is stored securely using environment variables and is **never hardcoded** in the source code.
