# Conesta App

## 📌 Project Description
Conesta App is a simple web application that displays a basic homepage for users. It demonstrates a working frontend setup using HTML.

## 🚀 Features
- Homepage with welcome message
- Simple UI structure
- Beginner-friendly project

## 🖥️ Core Screens
- Home Page (index.html)

## 🛠️ Tech Stack
- HTML

## ▶️ How to Run
1. Download or clone the repo
2. Open `index.html` in any browser
